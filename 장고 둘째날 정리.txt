템플릿 상속 .... {extends} 내
{block}  하위템플릿에서 재정의할 수 있는블록 정의
'''
{% extends "articles/base.html" %}

{% block content %}
  <h1>fake naver</h1>
  <form action="https://search.naver.com/search.naver/" method="GET">
    <label for="message">검색어:</label>
    <input type="text" name="query" id="message">
    <input type="submit">
  </form>
{% endblock content %}
'''
action : 입력데이터가 전송될 URL 지정(목적지), 없으면 현재 form의 페이지 url로 보내짐
method : 데이터를 어떤 방식으로 보낼지 정의. http request methods(GET, POST) 지정
-get post의 보안성 이길만큼 간편해서 사람들이 씀
-post
두 개 차이: 보안성
